# Street Runner – OpenGL Endless Runner

A 3D endless runner game built using C++ and OpenGL (GLUT).

## Features

- Procedural road generation
- DDA Line Algorithm
- Midpoint Circle Algorithm
- Dynamic difficulty scaling
- Day–Night cycle
- High score saving
- Pause and resume system
- Cartoon scenery (trees, windmill, houses, clouds, sun)

## Controls

- A / D → Move left / right
- SPACE → Jump
- ESC → Pause
- R → Restart

## Requirements

- C++
- OpenGL
- GLUT / FreeGLUT
